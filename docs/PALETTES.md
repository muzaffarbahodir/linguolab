# LinguoLab — палитры тем

Дизайн-система на 4 (макс 5) цветах на тему. Приглушённый текст = тот же
текст с прозрачностью (не отдельный цвет). Активная палитра — **Clay & Linen**.

Каждая палитра: `canvas` (фон) · `surface` (поверхность) · `ink` (текст) ·
`accent` (бренд) · `support` (опциональный 2-й акцент).

---

## ✅ 3 · Clay & Linen (активна)

Тёплая терракота + льняные нейтрали. Психология: тепло, дружелюбие,
мотивация, премиум-редактура — сильнее всего «хочется вернуться».

| Роль | Светлая | Тёмная |
|---|---|---|
| canvas | `#F4F1EA` | `#1B1815` |
| surface | `#FCFAF5` | `#251F18` |
| ink | `#2B2722` | `#ECE6DC` |
| accent (brand solid) | `#BE5F3C` | `#C8623F` |
| accent text (brand-400) | `#A8512E` | `#E59069` |
| support (teal) | `#4E8A82` | `#6FA89E` |
