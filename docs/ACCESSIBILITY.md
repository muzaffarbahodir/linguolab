# Доступность LinguoLab (Accessibility)

Документ для презентации клиенту. Описывает, какие меры доступности заложены
в приложение, чтобы им могли пользоваться **все** студенты — включая людей с
особенностями восприятия.

---

## 1. Поддержка дальтонизма (цветовая слепота)

~8% мужчин и ~0.5% женщин не различают некоторые цвета (чаще всего красный/зелёный).
Если статус показан **только цветом** — такой пользователь не поймёт разницу между,
например, «активным» и «отменённым» классом.

**Что сделано:**

Каждый статус в приложении передаётся **тремя независимыми каналами**, а не одним цветом:

| Канал | Пример | Зачем |
|-------|--------|-------|
| 🎨 Цвет | зелёный / красный / синий | быстрое распознавание для большинства |
| 🔣 Иконка | ▶️ идёт · ✅ завершён · ❌ отменён | различимо без цвета |
| 🔤 Текст | «Идёт», «Завершён», «Отменён» (на языке пользователя) | однозначно для всех |

Даже если человек вообще не видит цвета — иконка + текст полностью передают смысл.

**Где применяется:**
- Статусы классов (Скоро / Запись открыта / Идёт / Экзамен / Завершён / Отменён)
- Статусы платежей (Ожидает / Оплачен / Возврат / Ошибка …)
- Статусы пробных уроков (На рассмотрении / Подтверждено / Отменено)

Все статусы вынесены в единый модуль `lib/status.ts` — один источник, единый стандарт
по всему приложению.

---

## 2. Поддержка трёх языков (i18n)

Приложение полностью переведено на **русский, узбекский и английский**. Язык
интерфейса берётся из Telegram-профиля и переключается вручную. Весь текст —
включая статусы, кнопки, уведомления и сообщения об ошибках — идёт через систему
переводов, без «зашитого» русского.

---

## 3. Производительность на бюджетных устройствах

Многие студенты используют недорогие Android-телефоны. Приложение оптимизировано:
- Снижена нагрузка стеклянных (glass) эффектов → меньше «подтормаживаний» и расхода батареи.
- Поддержка системного режима «Уменьшить прозрачность»: при включении эффекты
  заменяются плотным фоном — интерфейс остаётся плавным даже на слабом железе.

---

## 4. Прочие меры

- **Тактильная отдача (haptic)** — отклик на нажатия, подтверждения, ошибки.
- **Размер зон нажатия** — кнопки не меньше 36px, удобно попадать пальцем.
- **Безопасные зоны** — интерфейс не залезает под «чёлку» и системные панели.
- **Альтернативный текст** у изображений — для программ чтения с экрана.
- **Скелетоны загрузки** — видно, что контент грузится, а не «завис».

---

_Эти решения делают LinguoLab доступным для максимально широкой аудитории и
соответствуют принципам инклюзивного дизайна (WCAG: «не полагаться только на цвет»)._
