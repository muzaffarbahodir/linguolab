# LinguoLab — Презентация приложения

**LinguoLab** — Telegram Mini App (TWA) для языкового учебного центра. Полный цикл: от пробного урока до сертификата. Работает внутри Telegram — установка не нужна, вход по Telegram-аккаунту.

- **Платформа:** Telegram WebApp (открывается из бота, кнопкой меню или ссылкой)
- **Языки интерфейса:** 🇷🇺 русский · 🇺🇿 oʻzbek · 🇬🇧 English (переключается на лету)
- **Валюты:** сум (UZS) и доллар (USD) — студент выбирает удобную
- **Дизайн:** тёмная «liquid glass» тема, плавные анимации, тактильный отклик (вибрация), поддержка спец-возможностей (уменьшение движения/прозрачности для слабого зрения и слабых телефонов)

---

## 👥 Роли пользователей

Одно приложение — шесть ролей. Интерфейс автоматически подстраивается под того, кто вошёл.

| Роль | Кто это | Что делает |
|------|---------|-----------|
| **Студент** | Ученик центра | Учится, записывается на курсы, платит, сдаёт ДЗ |
| **Родитель** | Родитель ученика | Следит за прогрессом ребёнка |
| **Учитель** | Преподаватель | Ведёт классы, отмечает посещаемость, проверяет ДЗ |
| **Менеджер** | Администратор-оператор | Заявки, зачисления, тикеты, студенты |
| **Админ** | Управляющий | Всё выше + финансы, рассылки, аналитика |
| **Супер-админ** | Владелец | Полный доступ, аудит, настройки оплаты |

Админ может **«примерить» любую роль** (предпросмотр), чтобы увидеть приложение глазами студента/учителя/родителя.

## 🎓 СТУДЕНТ — полный путь

### 1. Первый вход — Онбординг
3 экрана-слайда знакомят с приложением (учёба · расписание · достижения). Можно пропустить. Показывается один раз.

### 2. Главная (Home)
- **Приветствие** с аватаром и именем (утро/день/вечер — меняется по времени)
- **Прогресс** — общий процент прохождения с анимированной полосой
- **Ближайший урок** — язык, преподаватель, дата и время
- **Языки центра** — горизонтальная лента доступных языков
- **🎯 Пробный урок** — заявка в один тап: выбрать язык + комментарий → менеджер свяжется
- **Кнопка «Записаться»** + ⚡ быстрые действия (пробный урок / реферал / поддержка)

### 3. Определение уровня (Placement Test)
- Выбор языка → тест по нарастающей сложности (A1 → C2)
- Мгновенная обратная связь на каждый вопрос (верно/неверно)
- Результат: **уровень по шкале CEFR** (A1–C2) + балл + рекомендация
- Кнопка сразу записаться на подходящий курс

### 4. Запись на курс (Booking) — 3 шага
1. **Выбор языка** (плитки с флагами)
2. **Выбор класса** — название, преподаватель, цена/мес, свободные места, уровень
3. **Подтверждение** — карточка курса, расписание, цена → «Записаться» или «Оплатить сразу»

Если мест нет — запись в **лист ожидания** (автопродвижение, когда место освободится).

### 5. Курсы (Courses)
- Список открытых для записи и активных классов
- Фильтр по языку
- Карточка: статус (запись открыта / идёт / экзамен), уровень, преподаватель с аватаром и рейтингом, цена, обратный отсчёт до закрытия записи / конца семестра
- Запись или вход в лист ожидания
- Свои поданные заявки на пробные уроки со статусом

### 6. Расписание (Schedule)
- **Ближайшие уроки** из базы (дата, время, преподаватель, длительность)
- **Мои классы** — карточки с расписанием по дням недели + ближайшее занятие
- **Оценка преподавателя** — поставить звёзды 1–5 + комментарий (для активных классов)
- **Перевод в другой класс** — заявка с причиной
- **История посещаемости** — статистика % + список занятий с отметками (был / опоздал / пропустил / уважительно)

### 7. Оплата (Payment)
- Провайдеры: **Payme** 💳, **Click** 🔵 (расширяемо)
- Цена в выбранной валюте (UZS/USD)
- Защита от двойной оплаты (стабильный ключ идемпотентности)
- **История платежей** со статусами
- Счета по активным записям — «Оплатить» прямо из истории

### 8. Домашние задания (Homework)
- Фильтры: все / активные / сданные / просроченные (со счётчиками)
- Карточка: задание, дедлайн, статус, **оценка + комментарий преподавателя**
- **Сдача**: текстовый ответ и/или прикреплённый файл
- Сортировка: срочные сверху

### 9. Посещаемость (Attendance)
- Процент по каждому классу с цветовой шкалой (зелёный/жёлтый/красный)
- Разбивка: всего / был / опоздал / пропустил
- Напоминание о занятии (за 1 час)

### 10. Достижения (Achievements)
- Открытые и закрытые ачивки (дата получения)
- Геймификация прогресса

### 11. Сертификаты (Certificates)
- Полученные сертификаты по уровням (A1–C2)
- Открыть/скачать PDF

### 12. Реферальная программа
- Личный код-приглашение (копируется в один тап)
- Статистика: сколько пригласил, сколько бонусных дней получил
- Ввод чужого кода (взаимный бонус)

### 13. Приглашение родителя
- Сгенерировать код → родитель вводит его → связь установлена
- Родитель видит прогресс ребёнка

### 14. Поддержка (Support)
- Создать тикет (тема + сообщение)
- Список своих обращений со статусами (открыт / в работе / закрыт)

### 15. Уведомления (Notifications)
- Напоминания об уроках, новые ДЗ, оценки, подтверждения оплаты, рассылки
- Счётчик непрочитанных, отметить всё прочитанным

### 16. Профиль (Profile)
- Аватар, имя, роль, колокольчик уведомлений
- **Статистика**: прогресс, уроки, ДЗ, стрик 🔥, средняя оценка / посещаемость, уровень CEFR
- Реферальная карточка
- Быстрые ссылки на все разделы
- Настройки: валюта, язык интерфейса, тест уровня, оплата, пригласить родителя

## 👨‍👩‍👧 РОДИТЕЛЬ

### Кабинет родителя (ParentHome)
- Список детей (несколько детей — поддерживается)
- Карточка ребёнка: аватар, активные классы, дата привязки
- **Добавить ребёнка** — по коду-приглашению от ученика
- Колокольчик: уведомления о ребёнке (пропуск, новое ДЗ, оценка)

### Детальная страница ребёнка (3 вкладки)
**Обзор:**
- **3 кольцевых графика** (donut): посещаемость · средняя оценка · выполнение ДЗ
- Посещаемость за текущий месяц
- **Рейтинг в классе** 🥇🥈🥉 (место из общего числа)
- **Рейтинг по уровню** (среди всех студентов уровня)
- **Уровень CEFR** с прогресс-шкалой A1→C2
- **Преподаватели** ребёнка с рейтингом (переход в профиль)
- Ближайший урок · последнее ДЗ со статусом

**Посещаемость:** общий % + список занятий с отметками

**Домашние задания:** список с оценками и комментариями преподавателя

Можно **отвязать** ребёнка.

## 👨‍🏫 УЧИТЕЛЬ

### Кабинет (TeacherHome)
- Приветствие + дата
- **Уроки сегодня** — карточки с прогрессом посещаемости + «Начать урок»
- **ДЗ на проверке** — бейдж с количеством, быстрый список
- **Мои классы** — компактный список (студенты/мест, расписание)
- Кнопки: заявки на курсы 📋, статистика 📊, редактировать профиль ✏️

### Класс (TeacherClass) — 3 вкладки
**Уроки:**
- **Автогенерация** уроков на N недель по расписанию (2/4/8/12)
- Добавить урок вручную (дата, время, название)
- Список уроков со статусами → переход к отметке посещаемости

**Студенты:**
- Таблица: студент · посещаемость % · сданные ДЗ
- Переход в карточку студента (та же аналитика, что видит родитель)

**Домашние задания:**
- Создать ДЗ (название, описание, дедлайн)
- Список ДЗ → переход к проверке работ

### Отметка посещаемости (Attendance)
- Список студентов, статус каждого: был / опоздал / пропустил / уважительно
- «Все присутствуют» в один тап
- Сохранение всей группы разом

### Проверка работ (Submissions)
- Список сданных работ (текст + файл)
- Выставить **оценку (0–100) + комментарий**
- Открыть прикреплённый файл

### Статистика (TeacherStats)
- Классы · студенты · уроки · посещаемость · проверенные ДЗ
- Разбивка по каждому классу с прогрессом студентов

### Профиль преподавателя (публичный)
- Аватар, биография, **уровень** (по числу оценок), соцсети
- **Рейтинг** ⭐ со звёздной гистограммой + последние отзывы
- **Бейджи/достижения** (награждает админ)
- Список классов с записью
- Студенты ставят оценку преподавателю

### Заявки на курсы (ClassRequests)
- Учитель **предлагает открыть новый курс**: язык, название, уровень, расписание, мест, описание
- Видит статус (на рассмотрении / одобрено / отклонено) + комментарий админа
- При одобрении — курс создаётся автоматически

### Редактирование профиля
- Био, сайт, Instagram, Telegram

## 🛠 МЕНЕДЖЕР / АДМИН / СУПЕР-АДМИН

Вход в админ-панель — из профиля (баннер) или нижней навигации.

### Дашборд (AdminHome)
- **Виджеты** (Админ+): студенты, активные записи, уроки за неделю, преподаватели, ДЗ на проверке, выручка за месяц
- **Уведомления-алерты**: ожидающие переводы / пробные / тикеты / неактивные пользователи
- Плитки разделов
- **Предпросмотр ролей** — открыть приложение как студент/учитель/родитель/менеджер

### Пользователи (AdminUsers) — Админ+
- Вкладки: ожидают активации / все
- Фильтр по роли
- **Активация** новых пользователей
- **Смена роли** (студент → учитель → менеджер → …)

### Студенты (AdminStudents)
- Поиск, список, экспорт в **CSV** (Админ+)

### Преподаватели (AdminTeachers)
- Создать / редактировать / удалить
- **Награждение бейджами** (эмодзи + название + описание)

### Классы (AdminClasses)
- Создать / редактировать класс (язык, преподаватель, уровень, цены UZS+USD, мест)
- **Жизненный цикл семестра**: Черновик → Запись открыта → Активен → Экзамен → Завершён (→ Отменён)
- Настроить **расписание** (дни недели, время, длительность)
- Архив / удаление
- Переход к заявкам учителей на курсы

### Заявки на курсы (AdminClassRequests)
- Заявки от учителей
- **Одобрить** → задать цены, семестр, даты записи и семестра → курс создаётся
- **Отклонить** с комментарием

### Зачисления (AdminEnrollments)
- Вкладки по статусу
- Подтвердить (PENDING→ACTIVE) / отчислить / восстановить

### Переводы (AdminTransfers)
- Заявки на перевод между классами
- Одобрить / отклонить с комментарием, проверка свободных мест и комиссии

### Пробные уроки (AdminTrials)
- Заявки на пробные уроки → подтвердить / отменить

### Сертификаты (AdminCertificates)
- Выдать сертификат: по классу (список студентов) или глобальным поиском студента

### Поддержка (AdminSupport)
- Тикеты студентов → в работу / закрыть / переоткрыть

### Рефералы (AdminReferrals)
- Сводка: созданные коды, использованные, конверсия
- Топ пригласивших

### Финансы (AdminFinance) — Админ+
- Выручка за месяц / 6 мес, активные записи, всего студентов
- Графики выручки и новых студентов по месяцам
- Экспорт платежей в **CSV**

### Аналитика (AdminAnalytics) — Админ+
- Выручка и новые студенты (период 3/6/12 мес, столбчатые графики)
- **Воронка зачислений**: активные / ожидают / отчислены с процентами

### Рассылка (AdminBroadcast) — Админ+
- Сообщение всем пользователям в Telegram (до 4096 символов) с подтверждением

### Журнал аудита (AdminAudit) — Админ+
- Все действия: удаления, создания, смена ролей, активации, рассылки — кто и когда

### Настройки оплаты (AdminPaymentSettings) — Админ+
- Включить/выключить провайдеров (Payme / Click / Uzum), порядок отображения

## 🔐 Безопасность и надёжность

- Вход только через подпись Telegram (проверка HMAC initData)
- Доступ по ролям на каждом действии (guard'ы)
- Блокировка неактивных пользователей
- Защита платежей: проверка подписи провайдера (fail-closed), сверка суммы, идемпотентность (нет двойных списаний)
- Webhook'и Payme/Click создают/активируют запись автоматически после оплаты

## ⚙️ Технологии

- **Frontend:** React + Vite + Tailwind CSS, Telegram WebApp SDK
- **Backend:** NestJS (Node.js), PostgreSQL, Prisma ORM, Redis
- **Платежи:** Payme, Click (+ OFD Soliq для фискализации)
- **Деплой:** Docker, GitHub Actions CI/CD (автодеплой при пуше)
- **i18n:** react-i18next (3 языка)
