# LinguoLab

Платформа языкового центра на базе **Telegram Mini App**.

Студенты записываются на курсы, проходят пробные уроки и оплачивают занятия прямо в Telegram — без скачивания отдельных приложений.

## Возможности

- 📚 Каталог курсов с фильтрацией и поиском
- 🎓 Онлайн и офлайн пробные уроки
- 💳 Оплата через Payme и Click (UZS)
- 👨‍🏫 Личный кабинет учителя — уроки, домашние задания, расписание
- 👨‍👩‍👧 Кабинет родителя — прогресс ребёнка
- 📊 Админ-панель с аналитикой и управлением
- 🔔 Уведомления через Telegram бот

## Стек

| Часть | Технологии |
|---|---|
| Фронтенд (TWA) | React, Vite, Tailwind CSS |
| API | NestJS, Prisma, PostgreSQL, Redis |
| Админка | Next.js, NextAuth |
| Хранилище | Cloudflare R2 |
| Инфраструктура | Docker, GitHub Actions, Nginx |

## Запуск

```bash
pnpm install
pnpm dev
```

## Деплой

Push в `main` → GitHub Actions автоматически собирает образы и деплоит на сервер.
